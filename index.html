    Project maintained by martin-green Hosted on GitHub Pages — Theme by mattgraham

    Välkommen till Portalen.

    Det är ett spel baserat på den hydrauliska portalen som finns på Härsjösand (Masthugget-Majornas scoutkår). Du ska öppna för sugbilen så att den kan komma in och tömma trekammarbrunnen. Men den måste vara stängd för alla andra så att dom inte kör in och gör märken på gräsmattan.
